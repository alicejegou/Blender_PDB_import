# Blender_PDB_import
This is a python script to use in Blender (v.4.3.0) software command line in order to import .PDB file (3D proteins structures) 

<img width="350" height="550" alt="image" src="https://github.com/user-attachments/assets/733fbdab-b19b-407d-958a-647e6eb091a9" />

You can change atoms meshes base radius (EX: `bpy.ops.mesh.primitive_uv_sphere_add(radius=2.0)`)

<img width="350" height="550" alt="image" src="https://github.com/user-attachments/assets/f7efa57c-5566-4d8e-b581-e1cf31f2c99e" />

You can change atoms random scaling (EX: `scale = random.uniform(0.5, 1.5)`)

<img width="350" height="550" alt="image" src="https://github.com/user-attachments/assets/efb844ed-6f9a-4f90-8d74-f624f62bba7a" />

Once your atoms loaded, we can easily join them together and get unique protein object to manipulate
